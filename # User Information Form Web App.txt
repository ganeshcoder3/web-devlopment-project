# User Information Form Web App

## 1. Problem Statement

In many web applications, collecting user information such as personal details, preferences, and contact information is a core requirement. However, beginner projects often lack proper structure, validation, and user-friendly design, making forms difficult to use and understand.

**Objective:**
Design and implement a responsive, user-friendly **User Information Form** that collects personal details and user preferences using modern UI practices, client-side validation, and instant feedback—without requiring a backend.

---

## 2. Proposed Solution

A frontend-only web application that:

* Uses **Bootstrap** for responsive and clean UI
* Collects user credentials and preferences
* Validates inputs on the client side
* Displays submission results dynamically using JavaScript

This solution is ideal for learning form handling, UI design, and DOM manipulation.

---

## 3. Simple Architecture Diagram

```
[ User Browser ]
        |
        v
[ HTML + Bootstrap UI ]
        |
        v
[ JavaScript Validation & Logic ]
        |
        v
[ Result Display (No Backend) ]
```

The application runs entirely in the browser.

---

## 4. Tech Stack

**Frontend:**

* HTML5 – page structure
* CSS3 – custom styling
* Bootstrap 5 – responsive design
* JavaScript – form handling & DOM manipulation

**Backend:**

* Not required (Frontend-only project)

---

## 5. Setup Instructions

### Prerequisites

* Modern web browser (Chrome / Edge / Firefox)
* VS Code or any code editor
* Internet connection (for Bootstrap CDN)

### Steps

1. Create a project folder:

   ```
   user-info-form
   ```
2. Create files:

   ```
   index.html
   userinfo.css
   ```
3. Paste the HTML code into `index.html`
4. Paste the CSS code into `userinfo.css`
5. Open `index.html` in a browser

---

## 6. Prompt Strategy Summary

When using AI assistance during development, the prompt strategy followed was:

* Clearly specify required fields and UI framework (Bootstrap)
* Request simple, readable, and modular code
* Enhance UI first, then add JavaScript logic
* Test each change directly in the browser

This ensures correctness, clarity, and beginner-friendly output.

## 7. Source Code

### index.html

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>User Info Form</title>
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
    <link rel="stylesheet" href="userinfo.css">
</head>
<body>
    <div class="container py-5">
        <h1 class="text-center mb-4">Mohan Reddy's Form</h1>
        <h1 class="text-center mb-4">User Information</h1>
        <form id="userForm" class="mx-auto">
            <div class="mb-3">
                <label class="form-label">Name</label>
                <input type="text" class="form-control" id="name" required>
            </div>
            <div class="mb-3">
                <label class="form-label">Email</label>
                <input type="email" class="form-control" id="email" required>
            </div>
            <div class="mb-3">
                <label class="form-label">Password</label>
                <input type="password" class="form-control" id="password" required>
            </div>
            <div class="mb-3">
                <label class="form-label">Mobile Number</label>
                <input type="tel" class="form-control" id="mobile" pattern="[0-9]{10}" required>
            </div>

            <div class="mb-3">
                <label class="form-label">Favorite Cricketer</label>
                <div class="form-check"><input class="form-check-input" type="radio" name="cricketer" value="Virat Kohli"> Virat Kohli</div>
                <div class="form-check"><input class="form-check-input" type="radio" name="cricketer" value="MS Dhoni"> MS Dhoni</div>
                <div class="form-check"><input class="form-check-input" type="radio" name="cricketer" value="Rohit Sharma"> Rohit Sharma</div>
                <div class="form-check"><input class="form-check-input" type="radio" name="cricketer" value="Sachin Tendulkar"> Sachin Tendulkar</div>
            </div>

            <div class="mb-3">
                <label class="form-label">Favorite Movie Star</label>
                <div class="form-check"><input class="form-check-input" type="radio" name="movieStar" value="Allu Arjun"> Allu Arjun</div>
                <div class="form-check"><input class="form-check-input" type="radio" name="movieStar" value="Vijay Deverakonda"> Vijay Deverakonda</div>
                <div class="form-check"><input class="form-check-input" type="radio" name="movieStar" value="Mahesh Babu"> Mahesh Babu</div>
                <div class="form-check"><input class="form-check-input" type="radio" name="movieStar" value="NTR"> NTR</div>
                <div class="form-check"><input class="form-check-input" type="radio" name="movieStar" value="Nani"> Nani</div>
            </div>

            <button type="submit" class="btn btn-primary w-100">Submit</button>
        </form>
        <div id="formResult" class="mt-4 text-center"></div>
    </div>

    <script>
        document.getElementById('userForm').addEventListener('submit', function(e) {
            e.preventDefault();
            const name = document.getElementById('name').value;
            const cricketer = document.querySelector('input[name="cricketer"]:checked')?.value || 'None';
            const movieStar = document.querySelector('input[name="movieStar"]:checked')?.value || 'None';

            document.getElementById('formResult').innerHTML =
                `<div class='alert alert-success'>Thank you, <b>${name}</b>!<br>Your details have been submitted.<br><b>Favorite Cricketer:</b> ${cricketer}<br><b>Favorite Movie Star:</b> ${movieStar}</div>`;

            this.reset();
        });
    </script>
</body>
</html>
```
